# Instruções
1. Preencher o .env com base no .env.example, atentando-se em particular aos diretórios que serão montados
2. Preencher o htpasswd se forem criados usuários adicionais (usando o utilitário htpasswd do apache)
3. Preencher o .msmtprc no home com base no .msmtprc.example com os parâmetros para envio de e-mail
4. Criar no home as chaves de SSH relevantes
5. Depois de subir o sistema, configurar os hosts
